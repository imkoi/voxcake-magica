# VoxCake.Magica
VoxCake.Magica - Magica Voxel file reader for C#.
    
## Getting Started
1. Get latest [release](https://github.com/imkoi/voxcake-magica/releases/tag/1.0) or built it yourself from source.
2. Add reference to assembly in your project.
3. Use it!
